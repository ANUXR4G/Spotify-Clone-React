# Install NPM
```bash
npm install
```

# install tailwind in project
```bash
1. npm install -D tailwindcss 
2. npx tailwindcss init
```

# Tailwindconfig.js
``` bash
/** @type {import('tailwindcss').Config} */ 
module.exports = { 
  content: ["./src/**/*.{html,js}"], 
  theme: { 
    extend: {}, 
  }, 
  plugins: [], 
}  
```

# Add The Code Given Below In Index.css
```bash
@tailwind base; 
@tailwind components; 
@tailwind utilities;  
```
